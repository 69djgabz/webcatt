# Chat Real Time Socket.Io - Node - Express - MongoDB

Simple Chat using Socket.io

## Instalation

Install all dependencies

```
npm install
```

Run app

```
npm start
```

## Built With

- Node.Js
- Express.Js
- MongoDB
- Socket.IO
